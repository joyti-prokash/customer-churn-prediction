# Customer Churn Prediction

## Project Overview
This project predicts whether a customer will churn using machine learning models.

## Steps
- Data preprocessing
- Feature encoding
- Model building (Logistic Regression, Random Forest)
- Evaluation and threshold tuning

## Results
- Logistic Regression: ~75% accuracy, high recall
- Random Forest: ~79% accuracy
- Threshold tuning improved churn detection

## Key Insight
Recall is more important than accuracy in churn prediction.
